# Blender Studio 新片儿 -《奇点》
![img](https://hendasheng-web.oss-cn-beijing.aliyuncs.com/Weekly/vol.170/Blender-%E5%A5%87%E7%82%B9.webp)


[《奇点》](https://studio.blender.org/projects/singularity/)的故事发生在遥远的宇宙远古时期，那时恒星和星系都还很年轻。一场不祥的宇宙事件吞噬了一只小型太空生物的家园，它必须在一群同伴中寻找庇护，并受到一位神秘天体的守护。

![img](https://hendasheng-web.oss-cn-beijing.aliyuncs.com/Weekly/vol.170/Blender-%E5%A5%87%E7%82%B9-01.webp)

这部由安迪·戈拉尔奇克执导的短片，使用当时仍在开发中的最新版Blender制作而成。在制作《奇点》的过程中，Blender工作室团队扩展了[笔触工具库](https://extensions.blender.org/add-ons/brushstroke-tools/)，添加了新的绘画素材，并探索了几何节点在动画群体行为方面的程序化功能。

# Google Labs - DESIGN.md
![img](https://hendasheng-web.oss-cn-beijing.aliyuncs.com/Weekly/vol.170/google-design-md.png)

[DESIGN.md](https://github.com/google-labs-code/design.md) 是由 [谷歌实验室](https://labs.google/) 开源的 Agent 设计工具，可以简单理解为是给 AI / coding agent 读的设计系统说明书格式。

通过这种方式向编码代理描述视觉标识的格式规范，它使代理能够持久、结构化地理解设计系统。
