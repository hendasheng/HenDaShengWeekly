# Blender 和 Unreal
Blender 对 UE 的支持非常友好，可以很方便的交接各种文件，前几期提到过 Blender 输出 Gltf 输出到 UE 很好用，可总是要有导出、导入的步骤，其实 UE 很早就开发了相关 Blender 插件，在内部就可以将文件发送到 UE 中。
